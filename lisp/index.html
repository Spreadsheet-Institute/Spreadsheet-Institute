Home of Spreadsheet Lisp (SL)



Spreadsheet Lisp (SL) is a pure subset of the Lisp programming paradigm designed to simplify spreadsheet formulas. By aiming for "single-function formulas in every cell", Spreadsheet Lisp (SL) abstracts away unnecessary formula complexity in order to: improve formula readability, reduce formula sizes, minimize formula typos/mistakes, and multiply formula workloads.



How to use SL in your formulas:


Microsoft Excel:

  1. Download/save 0.9.0-xl.sl
  2. Run SpreadsheetLisp.vba
  3. Select 0.9.0-xl.sl file

Google Sheets:

  1. Download/save 0.9.0-gs.sl
  2. Run SpreadsheetLisp.gs
  3. Select 0.9.0-gs.sl file

If an error occurs, please email dev@spreadsheet.institute for support. (Or open an issue on GitHub!)
