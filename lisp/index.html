Instructions

  1. Download 0.9.0.sl
  2. If Microsoft Excel:
    run SpreadsheetLisp.vba
  3. If Google Sheets:
    run SpreadsheetLisp.gs
  4. Select "0.9.0.sl" download file
  5. Use SL functions directly in your formulas
