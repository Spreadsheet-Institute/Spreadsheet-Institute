Spreadsheet.Institute/Lisp

"Single-function formulas."



Spreadsheet Lisp (SL) is a pure subset of the Lisp programming paradigm designed to simplify spreadsheet formulas.

By aiming for "single-function formulas", Spreadsheet Lisp (SL) abstracts away unnecessary formula complexity in order to:

improve formula readability,
reduce formula sizes,
minimize formula typos/mistakes, and
multiply formula workloads.



How to use SL in your formulas:


Microsoft Excel


Pre-installed:

  • SpreadsheetLisp.xlsx

Install it yourself:

  1. Download/Save 0.9.0.sl
  2. Open any Microsoft Excel workbook (.xlsx/.xlsm)
  3. Click the Developer tab > Visual Basic
  4. From the top menu, click Insert > Module
  5. Copy SpreadsheetLisp.vba into the new module window
  6. Save the macro (Ctrl+S on PC / Command+S on Mac)
  7. Back in the workbook, click Developer > Macros > SpreadsheetLisp > Run
  8. Select the downloaded Spreadsheet Lisp (0.9.0.sl) source file

Google Sheets


Pre-installed:

  • docs.google.com

Install it yourself:

  1. Open any Google Sheet
  2. Click Extensions > Apps Script
  3. Copy/paste 0.9.0.js into Apps Script
  4. Click Save

If any error(s) occur(s): please email dev@spreadsheet.institute for support.
