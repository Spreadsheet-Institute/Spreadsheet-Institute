Editor de Definiciones



Lispa Hoja (LH) es un superconjunto de funciones de hoja de cálculo diseñadas para simplificar las fórmulas de la hoja de cálculo.

Al abordar (de manera asintótica) "fórmulas de función única", Lispa Hoja (LH) abstrae la complejidad innecesaria de las fórmulas para:

reducir las longitudes de las fórmulas, para
mejorando así la legibilidad de las fórmulas, lo que
minimiza los errores tipográficos y
multiplica las cargas de trabajo de las fórmulas.
